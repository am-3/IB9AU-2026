# Generative AI & AI Applications (IB9AU-2026)
This repository contains a comprehensive suite of technical tasks focused on the application of **Generative AI, Large Language Models (LLMs), and Advanced Machine Learning** within the financial and technology sectors. Each task is documented in a standalone Jupyter Notebook, emphasizing code efficiency, rigorous financial math, and critical interpretation of results.

## 🚀 Key Learning Objectives

* **Multimodal RAG:** Building Page-Wise Visual RAG systems for financial earnings reports using AstraZeneca FY2025 data.
* **Financial Modeling & Agents:** Developing autonomous agents using `smolagents` and `yfinance` to perform risk-adjusted performance analysis (Sharpe Ratios).
* **Generative Modeling:** Implementing Flow Matching, GANs, and CTGAN for synthetic data and floorplan generation.
* **VLM & OCR:** Utilizing Gemini Vision Language Models for automated document extraction and receipt verification.
* **AI Ethics & Audit:** Auditing LLM "Calendar Math" hallucinations and detecting "Risk Divergence" in 10-K filings for Block and PayPal.
